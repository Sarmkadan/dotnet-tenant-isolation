# Contributing to dotnet-tenant-isolation

We welcome contributions to improve this project.

### How to contribute
1. Fork the repo and create a feature branch.
2. Ensure you have the latest .NET SDK installed.
3. Make your changes, following existing code style.
4. Add unit tests for new features and verify with `dotnet test`.
5. Submit a Pull Request.

### Style Guide
- Follow Microsoft C# coding conventions.
- Use async/await for all I/O-bound operations.
- Include XML documentation for public APIs.
- Keep PRs focused on single features or bug fixes.

### Reporting Issues
Please provide:
- A clear description of the issue.
- Minimal reproducible example.
- .NET version and package versions.

We appreciate your help!

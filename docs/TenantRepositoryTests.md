# TenantRepositoryTests

Unit tests for the `TenantRepository` class, verifying tenant data access operations including retrieval by slug, status, and subscription expiry, as well as filtering and ordering behavior.

## API

### `TenantRepositoryTests`
Initializes a new instance of the test class with dependencies required for tenant repository testing, including a configured `TenantRepository` and test data context.
